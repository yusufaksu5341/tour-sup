# İkon Klasörü

Bu klasöre `app_icon.png` dosyası yerleştirilecektir.

Gereksinimler:
- Boyut: 1024x1024 piksel
- Format: PNG, şeffaf arka plan (iOS için)
- İçerik: TourSup logosu

Hazırlandıktan sonra çalıştırılacak komutlar:
```
dart run flutter_launcher_icons
dart run flutter_native_splash:create
```
